It preserves the separation between: mutable draft → reviewed claims/evidence → mutable package manifest → frozen snapshot → website/editorial handoff It does not approve evidence or claims, modify captures, approve mutable content, or publish from current database state. Approvals and releases always reference one immutable snapshot and manifest hash.
